## Flight Search Module

- Backend: Node.js + Express
- Database: MongoDB (Mongoose)
- No static JSON or external APIs used

### Features
- Flights seeded into MongoDB
- Search flights by departure and arrival city
- Maximum 10 flights returned per search
- Case-insensitive search
